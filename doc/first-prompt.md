# 依頼内容
「竹富町ごみ分別ガイド」というLINEチャットボットを紹介するための、モダンで清潔感のある静的ウェブサイト（シングルページ）を作成してください。

# 技術スタック（推奨）
- React + Tailwind CSS
- Lucide React (アイコン)
- レスポンシブ対応 (モバイルファースト)

# サイトの構成案
1. **ヒーローセクション**: 
   - キャッチコピー: 「島のごみを、もっと正しく、もっと楽に。」
   - サブコピー: 「最新AIが、24時間いつでもあなたの分別の悩みを解決します。」
   - LINE友だち追加ボタン (https://line.me/R/ti/p/@110rnxvm)
   
2. **AIチャットボットの特徴**:
   - 【即時回答】ゴミの名前を送るだけで、AIが分別方法を秒速回答。
   - 【高い推論能力】「これどう捨てる？」といった曖昧な質問も AI が理解。
   - 【最新データ】スプレッドシート連携により、常に最新の自治体ルールを反映。

3. **使いかた (How to use)**:
   - ステップ形式のイラストまたはアイコン付き説明。
   - ① LINEで友だち追加
   - ② ゴミの名前を送信（例：ペットボトル、モバイルバッテリー、使い捨てカイロ）
   - ③ AIが捨て方と収集日を回答

4. **透明性とカスタマイズ性 (For Admin)**:
   - 「データ管理の民主化」をアピール。
   - スプレッドシートからマスタデータを修正・更新できる仕組みを紹介。
   - [デモ用データを見る](https://docs.google.com/spreadsheets/d/1BYFBVFjkuNDHe0Z3uk1sHfROG7b1L9f21rkFcP4gJfo/edit?usp=sharing) へのリンク。

5. **開発者メッセージ**:
   - プロトタイプ版であることの免責事項。
   - 島の環境保護と、エンジニアリングによる課題解決の想い。

# デザインのトーン＆マナー
- ベースカラー: 清潔感のある白 (#FFFFFF) と、自然（西表島）を想起させるエメラルドグリーン (#059669)
- フォント: Sans-serif (読みやすさ重視)
- 全体的に丸みを帯びたポップなUIコンポーネントを使用し、親しみやすさを演出

# 出力物
- 全てのロジックとスタイルが含まれた1つのHTML/Reactコンポーネントコード
